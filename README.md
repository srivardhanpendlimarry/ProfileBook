# ProfileBook

A Social Media Platform built with Angular and ASP.NET Core Web API.

## Tech Stack
- Frontend: Angular 21
- Backend: ASP.NET Core Web API (.NET 8)
- Database: SQL Server with Entity Framework Core
- Authentication: JWT Token
- API Documentation: Swagger

## Features
- User Registration and Login
- Role-based Authentication (Admin/User)
- Create and View Posts (with Admin Approval)
- Like and Comment on Posts
- Search Users
- Report Users
- Admin Panel with CRUD operations
- Real-time messaging

## How to Run

### Backend
1. Open ProfileBook.API in Visual Studio
2. Update connection string in appsettings.json
3. Run migrations: Update-Database
4. Press F5 to run

### Frontend
1. Open ProfileBook.Frontend in VS Code
2. Run: npm install
3. Run: ng serve
4. Open: http://localhost:4200
